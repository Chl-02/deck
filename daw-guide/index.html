첫 마디를 누르고, 들어보고, 고른다.

Band AI DAW는 설명을 읽고 시작하는 도구가 아니라 마디를 눌러 바로 만드는 도구입니다. 코드, 기타 스트로크, 드럼 후보를 한 마디 단위로 들어보고 마음에 드는 것을 붙인 다음, 아래 피아노롤에서 MIDI를 직접 고칩니다.

처음 한 마디

처음에는 전체 곡을 만들려고 하지 말고 B1만 결정합니다. B1에서 괜찮은 조합이 나오면 B2로 넘어가고, 같은 식으로 4마디를 채우면 짧은 루프가 됩니다.

코드 후보는 마우스를 올리는 순간 미리듣기가 납니다. 기타 스트로크와 드럼은 후보 카드의 Listen으로 듣고, Select로 확정하거나 카드를 원하는 마디 위로 드래그합니다.

B1을 클릭

중앙 타임라인에서 Guitar, Bass, Drums, Piano 중 아무 트랙의 B1 칸을 누릅니다. 오른쪽에 B1 후보 패널이 열립니다.

Chord에서 코드 고르기

후보 카드에 마우스를 올려 짧게 들어봅니다. 마음에 드는 코드가 있으면 Select를 누릅니다.

Strum에서 손맛 고르기

다운 스트로크, 업 스트로크, 싱코페이션 느낌이 다른 후보를 Listen으로 비교합니다. 고른 패턴은 해당 마디의 기타 MIDI로 들어갑니다.

Drums에서 비트 고르기

킥, 스네어, 하이햇 배치가 다른 비트를 들어봅니다. 다른 파트와 맞는 후보를 고르면 그 마디의 드럼 그리드가 바뀝니다.

B2로 이동

Next를 누르거나 타임라인에서 B2를 클릭합니다. 코드를 유지할 수도 있고, 다음에 올 만한 코드 후보를 새로 골라도 됩니다.

화면 구조

01

상단 Transport

재생, 정지, 현재 위치, BPM, 키, 스냅, 렌더, 내보내기를 처리합니다. 스페이스 바도 여기와 같은 재생/정지 동작입니다.

02

중앙 Tracks

마디를 클릭하는 곳입니다. 이미 선택한 코드는 기타와 베이스 칸에, 선택한 비트는 드럼 칸에 짧게 표시됩니다.

03

오른쪽 후보 패널

현재 마디의 코드, 스트로크, 드럼 후보만 보여줍니다. 전체 곡 설명이 아니라 지금 누른 마디를 고치는 곳입니다.

피아노롤에서 직접 고치기

노트 이동

Select 모드에서 노트를 드래그하면 위치와 피치가 바뀝니다. 마디 후보로 만든 결과도 일반 MIDI처럼 고칠 수 있습니다.

길이 조절

노트 오른쪽 끝을 잡아 길이를 바꿉니다. 기타 스트로크가 너무 딱딱하면 짧은 노트를 약간 겹치거나 밀어 자연스럽게 만듭니다.

느낌 조절

하단 도구와 Inspector에서 velocity, timing feel, humanize 값을 조절합니다. 추천은 시작점이고 최종 결과는 직접 편집합니다.
