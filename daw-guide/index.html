처음 5분

1. 곡 조건 BPM, Key, Style을 정합니다.
2. 구간 Focus Section을 verse/chorus 등으로 고릅니다.
3. 코드 Chords에 `E - B - C#m - A`처럼 입력합니다.
4. 추천 Suggest 또는 Drum Recommend를 누릅니다.
5. 확인 Space로 듣고 MIDI/JSON으로 내보냅니다.

입력창에 포커스가 있을 때는 Space, Delete 같은 단축키가 작동하지 않습니다. 숫자나 코드를 편집한 뒤 작업면을 한 번 클릭하면 단축키가 다시 작동합니다.

추천 시스템 사용법

추천은 완성곡을 자동 생성하는 기능이 아니라, 현재 구간과 코드에 맞는 편곡 초안을 만들어 주는 기능입니다. 추천 결과는 반드시 타임라인과 하단 편집기에서 직접 수정할 수 있습니다.

전체 편곡 추천

  1. 왼쪽 Project에서 Focus Section을 선택합니다.
  2. Chords에 코드 진행을 입력합니다.
  3. 상단 BPM, Key, Style을 확인합니다.
  4. 상단 Suggest를 누릅니다.
  5. 타임라인에 생긴 추천 클립을 클릭합니다.
  6. Inspector에서 Apply 또는 Reject를 선택합니다.

드럼 비트 추천

  1. 하단 Drum Groove에서 Energy, Density, Sync, Fill을 조절합니다.
  2. Kick Lock으로 베이스/기타 악센트에 킥을 얼마나 맞출지 정합니다.
  3. Open Hat으로 오픈 하이햇 양을 정합니다.
  4. Recommend를 누르면 현재 Focus Section 마디에 드럼 그리드가 생성됩니다.
  5. K, Sn, HH, OH, Cr 셀을 클릭해 마디 단위로 수정합니다.
  6. Space로 듣고 Drum MIDI로 내보냅니다.

기타 스트럼 추천/렌더

  1. 상단 Suggest를 눌러 guitar_1 또는 guitar_2 추천 클립을 받습니다.
  2. 추천 기타 클립을 클릭하고 Apply합니다.
  3. 하단 Performance에서 Speed, Human, Swing, Mute, Accent를 조절합니다.
  4. Inspector의 Render를 누르면 WAV/MIDI가 생성됩니다.
  5. Inspector의 Preview 오디오 플레이어에서 기타 스트럼을 듣습니다.
컨트롤 영향
Focus Section 추천이 적용될 마디 범위를 결정합니다. 예: chorus는 17-24마디.
Chords 기타/피아노/베이스 편곡과 드럼 악센트 문맥의 기준이 됩니다.
Style 패턴 검색, 드럼 프로필, 기타 스트럼 성향에 영향을 줍니다.
Energy / Density 강도와 음표/히트 밀도를 조절합니다.
Kick Lock 드럼 킥을 베이스/다른 파트 악센트에 맞추는 정도입니다.

편집과 미리듣기

피아노 롤

  • Draw로 빈 칸을 클릭하면 노트가 추가됩니다.
  • Select로 노트를 드래그하면 위치와 피치를 바꿀 수 있습니다.
  • 노트 오른쪽 끝을 드래그하면 길이가 바뀝니다.
  • 선택 노트는 Pitch, Velocity, Length, Instrument 입력값으로도 수정할 수 있습니다.
  • MIDI 버튼은 현재 섹션의 편집 노트를 `.mid` 파일로 렌더합니다.

드럼 그리드

  • K는 킥, Sn은 스네어, HH는 클로즈 하이햇, OH는 오픈 하이햇, Cr은 크래시입니다.
  • 셀을 클릭하면 해당 히트가 켜지거나 꺼집니다.
  • 추천을 받기 전에도 빈 그리드에 직접 찍어서 미리듣기할 수 있습니다.
  • Drum MIDI 버튼은 현재 드럼 그리드를 MIDI로 렌더합니다.

미리듣기와 믹서

  • 상단 Play 또는 Space로 현재 루프 구간을 듣습니다.
  • Loop가 켜져 있으면 Focus Section만 반복 재생합니다.
  • Metro를 켜면 메트로놈 클릭이 같이 납니다.
  • M/S/R, Vol, Pan은 Web Audio 미리듣기에 반영됩니다.

산출물

버튼 결과
Export 프로젝트 JSON을 내려받습니다. 피아노 롤 노트, 드럼 그리드, 믹서, 루프/스냅 상태가 포함됩니다.
Piano Roll MIDI 현재 섹션의 피아노 롤 노트를 `data/processed/renders/piano_roll/`의 MIDI 파일로 렌더합니다.
Drum MIDI 현재 드럼 그리드를 `data/processed/renders/drums/`의 MIDI 파일로 렌더합니다.
Guitar Render 선택한 기타 클립의 MIDI와 WAV 미리듣기 파일을 생성합니다.

문제 해결

소리가 안 날 때

브라우저가 첫 사용자 입력 전 오디오를 막을 수 있습니다. 앱 화면을 한 번 클릭한 뒤 Space 또는 Play를 다시 누릅니다.

추천/렌더가 실패할 때

현재 백엔드는 임시 터널입니다. 로컬 백엔드나 Cloudflare 터널이 꺼지면 API 기능이 실패할 수 있습니다.

단축키가 안 먹을 때

입력창, 셀렉트, 숫자 필드에 포커스가 있으면 단축키를 의도적으로 무시합니다. 빈 작업면을 클릭한 뒤 다시 누릅니다.
