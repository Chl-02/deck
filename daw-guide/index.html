처음 10분: 코러스 8마디 만들기

아래 값은 앱 기본값과 거의 같습니다. 처음에는 창작하려고 하지 말고, 이 값 그대로 눌러서 “어디에 결과가 생기는지”부터 확인하세요.

실습 재료

모던 록 코러스 8마디. 코드 진행은 흔한 팝/록 진행으로 두고, 드럼과 기타를 먼저 만듭니다.

약 10분
BPM: 148 Key: E major Style: modern rock Focus Section: chorus Chords: E - B - C#m - A
1

앱을 열고 조건을 맞춥니다

  1. chl02.com/daw/를 엽니다.
  2. 상단에서 BPM을 148로 둡니다.
  3. Key는 E major, Style은 modern rock을 둡니다.
  4. 왼쪽 Focus Section을 chorus로 둡니다.
  5. Chords에는 E - B - C#m - A를 입력합니다.
결과: 앞으로 누르는 추천과 편집은 chorus, 즉 17-24마디 기준으로 움직입니다.
2

드럼을 먼저 만듭니다

  1. 하단 Drum Groove에서 Energy와 Density를 0.7-0.8 근처로 둡니다.
  2. Kick Lock은 0.7 이상으로 둡니다.
  3. Recommend를 누릅니다.
  4. 왼쪽 그리드의 K, Sn, HH 셀을 클릭해 한두 개만 바꿔봅니다.
  5. Space를 눌러 듣습니다.
결과: 추천된 드럼은 전체 곡이 아니라 현재 섹션의 마디 그리드에 들어갑니다. 바로 셀 단위로 고칠 수 있습니다.
3

기타 스트럼을 들어봅니다

  1. 타임라인에서 guitar_1 클립을 클릭합니다.
  2. 오른쪽 Inspector에서 Render를 누릅니다.
  3. Preview 오디오 플레이어가 나타나면 재생합니다.
  4. 오른쪽 아래 Performance의 Speed, Human, Swing, Mute, Accent를 바꿉니다.
  5. 다시 Render를 눌러 차이를 듣습니다.
결과: 기타는 코드가 동시에 나는 느낌이 아니라, 줄을 쓸어내리는 시간차와 세기 차이를 넣어 렌더합니다.
4

피아노 롤을 직접 고칩니다

  1. 하단 Piano Roll에서 Seed Chords를 누릅니다.
  2. Draw를 선택하고 빈 칸을 클릭해 노트를 하나 추가합니다.
  3. Select로 바꾼 뒤 노트를 드래그해서 위치나 높이를 바꿉니다.
  4. 노트 오른쪽 끝을 잡아 길이를 조절합니다.
  5. 선택한 노트는 아래 Pitch, Velocity, Length 숫자로도 고칩니다.
결과: 피아노 롤은 보기용이 아니라 실제 MIDI 노트 편집기입니다.
5

루프와 믹서를 맞춥니다

  1. 상단 Loop를 켜면 chorus만 반복해서 듣습니다.
  2. Metro를 켜면 클릭을 같이 듣습니다.
  3. 오른쪽 아래 Mixer에서 악기별 M, S, Vol, Pan을 조절합니다.
  4. 기타만 듣고 싶으면 guitar_1의 S를 누릅니다.
결과: 브라우저에서 바로 소리를 확인합니다. MIDI 파일을 내려받기 전에도 편곡 밸런스를 볼 수 있습니다.
6

필요한 파일을 뺍니다

  1. 상단 Export는 현재 프로젝트 JSON을 내려받습니다.
  2. 드럼만 MIDI로 필요하면 Drum MIDI를 누릅니다.
  3. 피아노 롤 노트만 MIDI로 필요하면 Piano Roll의 MIDI를 누릅니다.
  4. 기타는 클립 선택 후 Inspector Render로 WAV/MIDI를 만듭니다.
결과: DAW로 가져갈 수 있는 MIDI와 다시 열어볼 수 있는 JSON을 분리해서 얻습니다.

추천은 “완성곡 생성”이 아닙니다

이 앱의 추천은 현재 마디에 넣을 초안을 만드는 기능입니다. 핵심은 추천을 받은 뒤 곧바로 마디, 셀, 노트, 노브 단위로 고치는 것입니다.

Step 1

문맥을 정합니다

Focus Section, Chords, Style이 추천의 기준입니다. chorus를 고르면 17-24마디에 맞춰집니다.

Step 2

후보를 받습니다

상단 Suggest는 타임라인 클립 후보를 만들고, 드럼 Recommend는 드럼 그리드 후보를 만듭니다.

Step 3

채택 여부를 정합니다

타임라인 후보는 선택한 뒤 Apply로 채택합니다. 마음에 안 들면 Reject로 버립니다.

Step 4

손으로 고칩니다

드럼은 셀, 피아노는 노트, 기타는 Performance 노브로 고칩니다. 이 단계가 실제 편곡입니다.

LLM API는 필요 없습니다. 현재 드럼 추천, 기타 스트럼 렌더, 피아노 롤 편집, 브라우저 미리듣기는 규칙/모델/로컬 엔진 기반입니다. 다만 추천과 렌더 API는 백엔드가 살아 있어야 합니다. 백엔드가 꺼져 있으면 화면은 열리지만 Recommend/Render 계열이 실패할 수 있습니다.

드럼 비트 추천을 쓰는 법

드럼은 먼저 추천받고, 마음에 안 드는 킥/스네어/하이햇만 마디별로 고치는 방식이 가장 빠릅니다.

추천받기

  1. Focus Section을 고릅니다. 처음에는 chorus로 둡니다.
  2. Chords를 확인합니다. 드럼도 다른 파트의 악센트 문맥을 참고합니다.
  3. Energy는 세기, Density는 음표 수라고 생각하면 됩니다.
  4. Sync를 올리면 정박에서 조금 벗어난 느낌이 늘어납니다.
  5. Fill을 올리면 마디 끝 장식이 많아집니다.
  6. Recommend를 누릅니다.

마디별로 고치기

  1. 왼쪽 드럼 그리드에서 B17, B18처럼 마디를 확인합니다.
  2. K는 킥, Sn은 스네어, HH는 클로즈 하이햇입니다.
  3. OH는 오픈 하이햇, Cr은 크래시입니다.
  4. 셀을 클릭하면 켜지고, 다시 클릭하면 꺼집니다.
  5. 한 마디만 바꿔도 바로 Space로 들을 수 있습니다.
컨트롤 올리면 내리면
Energy 크래시, 강한 킥, 강한 스네어가 늘어납니다. 절제된 벌스 느낌에 가까워집니다.
Density 하이햇과 보조 히트가 촘촘해집니다. 공간이 많고 단순해집니다.
Kick Lock 킥이 베이스/기타 악센트에 더 붙습니다. 드럼이 독립적으로 움직입니다.
Open Hat 오픈 하이햇이 더 자주 나옵니다. 닫힌 하이햇 위주의 단단한 리듬이 됩니다.

기타 스트로크 추천과 보정

기타는 “패턴 선택 → Apply → Performance 노브 조절 → Render로 듣기” 순서입니다. 스트럼 보정은 코드 노트를 같은 시각에 찍는 것이 아니라, 실제로 손이 줄을 지나가는 시간차를 만드는 쪽에 가깝습니다.

추천 클립 받기

  1. 상단 Suggest를 누릅니다.
  2. 타임라인에 guitar_1 또는 guitar_2 후보가 생기면 클릭합니다.
  3. 오른쪽 Inspector에서 Energy, Density, Role을 봅니다.
  4. 쓸 만하면 Apply, 아니면 Reject를 누릅니다.

사람처럼 보정하기

  1. Speed는 한 번 쓸어내리는 시간입니다. 낮으면 빠르고 단단합니다.
  2. Human은 타이밍 흔들림입니다. 너무 높이면 헐거워집니다.
  3. Vel Rand는 줄마다 세기 차이입니다.
  4. Swing은 뒤쪽 8분/16분을 살짝 늦춥니다.
  5. Mute는 팜뮤트 느낌, Accent는 강세 차이입니다.

소리로 확인하기

  1. 기타 클립을 선택합니다.
  2. Inspector Render를 누릅니다.
  3. Preview 플레이어가 생기면 들어봅니다.
  4. 노브를 바꿨으면 다시 Render해야 새 보정이 반영됩니다.

피아노 롤 편집

피아노 롤은 다른 DAW처럼 노트의 위치, 높이, 길이, 세기를 직접 바꾸는 영역입니다. 여기서 만든 노트는 미리듣기와 MIDI 렌더에 반영됩니다.

마우스로 편집

  • Draw: 빈 칸을 클릭하면 노트를 추가합니다.
  • Select: 노트를 클릭하고 드래그하면 위치와 피치를 바꿉니다.
  • 오른쪽 끝 드래그: 노트 길이를 바꿉니다.
  • Duplicate: 선택한 노트를 복제합니다.
  • Delete: 선택한 노트를 지웁니다.

버튼과 숫자로 편집

  • Seed Chords: 현재 코드 진행으로 기본 노트를 다시 깔아줍니다.
  • Quantize: 현재 Snap 값에 맞춰 노트를 정렬합니다.
  • Instrument: 선택 노트를 piano, bass, guitar, vocal 중 하나로 바꿉니다.
  • Pitch: MIDI 피치 번호입니다. 값이 클수록 높습니다.
  • Velocity: 세기입니다. 1-127 범위입니다.
  • Length: 16분음표 단위 길이입니다.
피아노 롤에서 클릭했는데 아무것도 안 생기면 Draw가 켜져 있는지 확인하세요. 노트를 움직이고 싶은데 새 노트가 생기면 Select로 다시 바꾸면 됩니다.

처음에 자주 헷갈리는 것

!

Suggest를 눌렀는데 곡이 완성되지 않습니다

정상입니다. Suggest는 후보 클립을 만드는 기능입니다. 후보를 선택하고 Apply한 뒤 직접 수정해야 합니다.

!

Space가 안 먹습니다

입력창에 커서가 있으면 단축키가 꺼집니다. 빈 작업면을 클릭한 뒤 다시 누르세요.

!

Render나 Recommend가 실패합니다

백엔드 API가 꺼져 있거나 임시 터널이 바뀐 상태일 수 있습니다. 화면 미리듣기 일부는 되더라도 서버 렌더는 실패할 수 있습니다.
